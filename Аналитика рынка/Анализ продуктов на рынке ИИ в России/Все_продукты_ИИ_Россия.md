## 1. Большие языковые модели (LLM)

### GigaChat (Сбер)
**Идея:** Мультимодальная языковая модель от Сбера. Работает с текстом, кодом и изображениями. Предназначена как для массового пользователя (чат на giga.chat), так и для корпоративного сектора через API. Поддерживает функцию Deep Research для глубокого анализа информации. Выпущена в трёх версиях: Lite (быстрые задачи), Pro (баланс скорости и качества), Max (сложные задачи). В 2025 году вышла версия GigaChat 2, в 2026 — GigaChat 3.
**Техническая реализация:** Собственная архитектура на базе transformer. Обучена на русскоязычных и мультиязычных корпусах данных. Доступна через REST API в облаке Сбера. Поддерживает function calling, работу с контекстом до 32K токенов. On-premise вариант для корпораций.
**Ссылка:** https://giga.chat

### YandexGPT (Яндекс)
**Идея:** Языковая модель Яндекса, глубоко интегрированная в экосистему компании (Поиск, Алиса, Яндекс 360, DataSphere). Используется для генерации текста, черновиков документов, суммаризации, классификации. Ключевое преимущество — доступ к актуальной информации из поисковой системы Яндекса. В 2026 году вышла версия YandexGPT 5.
**Техническая реализация:** Собственная архитектура transformer. Обучена на русскоязычных данных с упором на качество русского языка. Доступна через Yandex Cloud API и Yandex AI Studio. Поддерживает дообучение (fine-tuning) на пользовательских данных.
**Ссылка:** https://cloud.yandex.ru/services/yandexgpt

### T-Pro 2.0 (Т-Банк)
**Идея:** Языковая модель от Т-Банка, заточенная на сложные инструкции и аналитические задачи. Способна к «размышлению» — пошаговому решению задач с промежуточными рассуждениями. Используется внутри экосистемы Т-Банка и доступна как открытая модель.
**Техническая реализация:** Открытая модель, доступна на Hugging Face. Архитектура transformer с поддержкой chain-of-thought reasoning. Оптимизирована для русского языка.
**Ссылка:** https://huggingface.co/T-Bank

### ruGPT (AI-Forever / Сбер)
**Идея:** Семейство открытых русскоязычных языковых моделей для локального развёртывания. Предназначены для разработчиков и компаний, которым нужна LLM без зависимости от облачных API. Подходит для задач, где данные не могут покидать контур компании.
**Техническая реализация:** Открытый исходный код, доступен на Hugging Face. Модели разного размера (от 1.3B до 13B параметров). Архитектура GPT (decoder-only transformer). Обучены на русскоязычных корпусах, включая книги, Wikipedia, новости.
**Ссылка:** https://github.com/ai-forever

### Kodify-Nano (МТС AI)
**Идея:** Специализированная модель от МТС для генерации и анализа программного кода. Ориентирована на разработчиков — помогает писать, рефакторить и отлаживать код. Компактная модель, оптимизированная для быстрой работы.
**Техническая реализация:** Compact transformer, обученный на корпусах кода. Поддерживает основные языки программирования. Оптимизирован для инференса на ограниченных ресурсах.
**Ссылка:** https://ai.mts.ru

### A-Vibe (Авито)
**Идея:** Внутренняя языковая модель Авито для работы с объявлениями — автоматическая классификация, поиск, генерация описаний товаров. Заточена под специфику маркетплейса: понимает категории, характеристики товаров, пользовательский сленг.
**Техническая реализация:** Собственная модель, обученная на данных Авито. Используется внутри платформы для улучшения поиска и рекомендаций.
**Ссылка:** https://www.avito.ru

### DeepPavlov (МФТИ / открытый проект)
**Идея:** Экосистема библиотек для разработки приложений на основе обработки естественного языка. Предоставляет готовые модели для русского языка: NER, классификация, ответы на вопросы, диалоговые системы. Активно используется разработчиками по всему миру.
**Техническая реализация:** Открытый исходный код на Python. Построен поверх PyTorch и Transformers. Включает предобученные модели RuBERT, Sentence-RuBERT для русского языка. Модульная архитектура для создания пайплайнов NLP.
**Ссылка:** https://deeppavlov.ai

### Vikhr (открытый проект) ⭐
**Идея:** Открытая русскоязычная языковая модель, разрабатываемая сообществом. Позиционируется как независимая альтернатива корпоративным LLM для русского языка.
**Техническая реализация:** Открытая модель на Hugging Face, основана на fine-tuning существующих архитектур под русский язык.
**Ссылка:** https://huggingface.co/Vikhrmodels

## 2. ИИ-копирайтинг и генерация текста

### Шедеврум (Яндекс)
**Идея:** Мультимодальная платформа от Яндекса для массового пользователя. Генерирует изображения, видео и тексты по запросу на русском языке. Доступна как мобильное приложение и веб-сервис. Социальная составляющая — пользователи делятся результатами.
**Техническая реализация:** Использует YandexGPT для текста и YandexART для изображений. Мобильное приложение на iOS/Android. Облачный инференс через инфраструктуру Яндекса.
**Ссылка:** https://shedevrum.ai

### Turbotext
**Идея:** Российский ИИ-сервис для генерации текстового контента: статьи, посты в соцсети, описания товаров, SEO-тексты. Ориентирован на копирайтеров, маркетологов и владельцев бизнеса, которым нужно быстро создавать контент на русском языке.
**Техническая реализация:** Веб-интерфейс, использует языковые модели с дообучением под копирайтинг-задачи. Шаблоны для разных типов контента.
**Ссылка:** https://turbotext.ai

### CopyMonkey
**Идея:** Сервис для генерации контента для маркетплейсов — карточки товаров, SEO-описания, заголовки. Заточен под Wildberries, Ozon и другие российские площадки. Понимает специфику категорий товаров и требования маркетплейсов.
**Техническая реализация:** Веб-платформа с API. Обучен на успешных карточках товаров российских маркетплейсов.
**Ссылка:** https://copymonkey.ai

### Fabula AI ⭐
**Идея:** ИИ-сервис для быстрого создания оформленных карточек товаров для маркетплейсов без дизайнера. Генерирует визуальное оформление и текстовые описания. Решает конкретную боль продавцов — быстрый запуск новых товаров на площадках.
**Техническая реализация:** Комбинация генеративных моделей для изображений и текста. Шаблоны адаптированы под требования Wildberries, Ozon.
**Ссылка:** https://fabula.ai

### Яндекс 360 с ИИ
**Идея:** Офисный пакет Яндекса (документы, таблицы, почта) со встроенным ИИ-ассистентом. Помогает писать письма, суммаризировать документы, генерировать тексты прямо внутри рабочего процесса.
**Техническая реализация:** YandexGPT интегрирован в веб-редакторы документов, почтовый клиент и другие сервисы Яндекс 360.
**Ссылка:** https://360.yandex.ru

## 3. Генерация изображений

### Kandinsky (Сбер / Kandinsky Lab)
**Идея:** Диффузионная модель для генерации изображений по текстовому описанию от Сбера. Одна из самых развитых российских моделей генерации — поддерживает text-to-image, image-to-image, inpainting. Доступна как для массовых пользователей, так и для бизнеса через API.
**Техническая реализация:** Архитектура латентной диффузии (Latent Diffusion). Обучена на масштабном мультиязычном датасете. Версии: Kandinsky 2.x, 3.x, 4.x с улучшением качества. Kandinsky Image Lite — облегчённая версия для быстрой генерации. Доступна через API GigaChat.
**Ссылка:** https://developers.sber.ru/portal/products/kandinsky

### YandexART (Яндекс)
**Идея:** Генеративная модель Яндекса для создания изображений и анимации по текстовому описанию. Интегрирована в экосистему Яндекса — доступна через Шедеврум, Алису и Yandex Cloud.
**Техническая реализация:** Собственная диффузионная архитектура. Доступна через Yandex Cloud API. Поддерживает русскоязычные промпты.
**Ссылка:** https://cloud.yandex.ru/services/yandexart

### Freestyling.ai ⭐
**Идея:** Российский стартап, объединяющий генерацию изображений, видео, ИИ-редактирование, удаление фона и смену формата в одной платформе. Позиционируется как «всё в одном» для создателей контента.
**Техническая реализация:** Веб-платформа с набором генеративных моделей. Комбинирует несколько ИИ-инструментов в одном интерфейсе.
**Ссылка:** https://freestyling.ai

### Flyvi
**Идея:** Графический редактор для создания контента со встроенными нейросетями. Сочетает готовые шаблоны, простые инструменты редактирования и ИИ-генерацию. Ориентирован на SMM-менеджеров, маркетологов.
**Техническая реализация:** Веб-приложение с canvas-редактором. Встроенные ИИ-модели для генерации и редактирования изображений.
**Ссылка:** https://flyvi.io

### A-Vision (Авито)
**Идея:** Внутренняя модель Авито для анализа изображений — автоматическая модерация контента, классификация фотографий в объявлениях, определение категории товара по фото.
**Техническая реализация:** Собственная модель компьютерного зрения, обученная на данных Авито.
**Ссылка:** https://www.avito.ru

## 4. Генерация видео

### Kandinsky Video (Сбер)
**Идея:** Генерация видео по текстовому описанию или стартовому кадру. Эволюция модели: Video 1.1 (до 6 секунд) → 4.0 (до 12 секунд в HD) → 4.1 (до 10 секунд). Поддерживает text-to-video и image-to-video.
**Техническая реализация:** Диффузионная архитектура для видео. Варианты качества: SD 720x576, HD 1280x720. Версии Lite и Pro для разных задач. Представлена на GigaConf 2025.
**Ссылка:** https://developers.sber.ru/portal/products/kandinsky

### Шедеврум Видео (Яндекс)
**Идея:** Генерация коротких видеороликов по текстовому описанию внутри платформы Шедеврум. Доступна массовому пользователю через приложение.
**Техническая реализация:** Основана на YandexART и собственных видео-моделях Яндекса.
**Ссылка:** https://shedevrum.ai

### GPTunnel Video
**Идея:** Российская платформа для создания видео с помощью ИИ — от клипов по тексту до презентационных видео. Закрывает разные сценарии использования.
**Техническая реализация:** Веб-платформа, агрегирует несколько моделей генерации видео.
**Ссылка:** https://gptunnel.ru

### Visper ⭐
**Идея:** Платформа для создания видео с цифровыми аватарами. Генерирует реалистичные видео с говорящим человеком по тексту — для обучения, маркетинга, презентаций. Не нужна камера, студия или актёры.
**Техническая реализация:** Синтез видео с lip-sync на базе нейросетей. Библиотека готовых аватаров. Поддерживает русский язык.
**Ссылка:** https://visper.tech

## 5. Генерация музыки и аудио

### Звук Студио (Сбер / Звук)
**Идея:** ИИ-инструмент для музыкантов, встроенный в платформу «Звук». Генерирует сэмплы, музыкальные заготовки и мелодии по текстовому описанию. Помогает ускорить творческий процесс — быстро набросать идею трека.
**Техническая реализация:** Генеративная модель для аудио, обученная на музыкальных данных. Интегрирована в платформу Звук.
**Ссылка:** https://zvuk.com

### Mubert
**Идея:** Российский стартап для генерации фоновой музыки с помощью ИИ. Создаёт бесконечные уникальные треки для видео, подкастов, стримов, приложений. Музыка генерируется в реальном времени и не требует лицензирования.
**Техническая реализация:** Генеративная модель, комбинирующая звуковые сэмплы в реальном времени. API для интеграции в приложения. Модель обучена на библиотеке лицензированных сэмплов.
**Ссылка:** https://mubert.com

## 6. Голосовые ассистенты

### Алиса (Яндекс)
**Идея:** Голосовой ассистент Яндекса с генеративными ИИ-функциями. Работает в умных колонках (Яндекс Станция), приложениях, браузере. Умеет вести свободный диалог, отвечать на вопросы, управлять умным домом, играть музыку. С 2024 года получила генеративные возможности на базе YandexGPT.
**Техническая реализация:** YandexGPT + Yandex SpeechKit для голоса. Мультимодальная система: STT → LLM → TTS. Интеграция со всеми сервисами Яндекса.
**Ссылка:** https://alice.yandex.ru

### Салют (Сбер)
**Идея:** Семейство виртуальных ассистентов Сбера (Сбер, Афина, Джой) с разными «характерами». Работают в приложении СберБанк Онлайн, умных устройствах SberBox/SberPortal, на ТВ. Управление банковскими операциями голосом, умный дом, развлечения.
**Техническая реализация:** GigaChat + SaluteSpeech. Мультимодальная система с поддержкой жестов на устройствах с камерой.
**Ссылка:** https://salute.sber.ru

### Маруся (VK)
**Идея:** Голосовой ассистент VK. Работает в умных колонках VK Капсула, приложении VK. Умеет отвечать на вопросы, управлять музыкой VK, умным домом, играть в игры.
**Техническая реализация:** Собственные модели VK для NLU и генерации. Интеграция с экосистемой VK.
**Ссылка:** https://marusia.vk.com

## 7. Синтез речи (TTS)

### SaluteSpeech (Сбер)
**Идея:** Облачный сервис синтеза и распознавания речи от Сбера. Генерирует естественную речь на русском языке с разными голосами и интонациями. Используется в колл-центрах, IVR-системах, приложениях.
**Техническая реализация:** Нейросетевой TTS на базе собственных моделей. Доступен через API. Поддерживает SSML-разметку для управления интонацией.
**Ссылка:** https://developers.sber.ru/portal/products/salutespeech
